# HART Hub Staff Toolkit

**Kenora Area HART Hub — Staff Onboarding & Resource Portal**

A comprehensive, single-page web toolkit for HART Hub staff covering intake, triage, referral pathways, service coordination, and culturally responsive care delivery.

> **OGIMAAWABIITONG · Kenora Chiefs Advisory (KCA)**

---

## What's Inside

| Section | Description |
|---------|-------------|
| **Staff Roles** | Funded positions across KCA, CMHA-K, WNHAC, KACL |
| **Triage & Assessment** | Safety screen, needs assessment, consent, tier assignment |
| **Referral Matrix** | Priority rules, sequencing, access site routing |
| **Addiction Services** | In-scope vs. out-of-scope, RAAM, detox, community agencies |
| **Supportive Housing** | Norwood, KDSB coordinated access, eligibility |
| **Case Management** | CMHAK, KACL, KCA service navigation |
| **Forms & Consent** | Consent form, information sharing, documentation |
| **Process Visuals** | Pathway diagrams, integration models, system maps |

## Features

- Single self-contained HTML file — no build step, no dependencies
- Full-text search across all content
- Tabbed navigation within each section
- Click-to-zoom process diagrams
- Mobile responsive
- Print-friendly
- Works offline once loaded
